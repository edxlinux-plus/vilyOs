Instalação do Xubuntu-Core

Para instalar o xubuntu-core, utilize o seguinte comando:

apt install xubuntu-core

Se já estiver instalado, não é necessário reinstalar.

---

Modificar a Aparência

1. Baixar xubuntu-core.tar.gz

Faça o download do arquivo xubuntu-core.tar.gz através do link:

https://github.com/edxlinux-plus/vilyOs/blob/main/xubuntu-core.tar.gz

2. Copiar para o sistema

Copie o arquivo para o diretório /usr/share. Navegue até ele com permissão de root no terminal:

cd /usr/share
cp xubuntu-core.tar.gz /usr/share

3. Descompactar xubuntu-core.tar.gz

Descompacte o arquivo com o comando:

tar -vzxf xubuntu-core.tar.gz

4. Remover xubuntu-core.tar.gz

Após descompactar, você pode remover o arquivo compactado:

rm xubuntu-core.tar.gz
